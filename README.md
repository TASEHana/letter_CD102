
# How to download 
```
git clone git@github.com:TASEHana/letter_CD102.git
cd letter_CD102
```
# How to compile

## Download the zipped revtex4 file and copy styles
```
./copyStyle
``` 

## compile with the following script
```
./makePDF letter
```

## Output 
The produced PDF file is called main.pdf
